Pseudo - Elements:

A Pseudo - Elements lets you style specific parts of an element.

Syntx: selector::Pseudo-elements {P:values}

::before - inserts content before elements content - add icons,shapes

::after - inserts content after elements's content. - deccarative elements

::first-letter - styles first letter - articles - drop-cap style

::first-line - styles first line:

::selection styles text when user selectes it.

::marker - styles list markers (bullets/number/dot)


Animations:
CSS animations are mainly done with transition for simple state changes and @keyframes with animations-* properties for complex, repeated or timed effects.

Transition - smooth change of property over the time.

Transform  - moves,rotates,scales,skews elements

@keyframes - defines animation steps.

animations-name - Name

animation-duration - time

animation-* - (duratio,iteration-cout,direction,fill-mode)



Media Query:
1. A media query is a feature of css that allows web content to adapt  to different screen size and device types like (phone,tablets,laptops,desktops).
2. media queries help create responsive web designs, which means your site looks good on all devies.

media queries check for certain conditions (like screen width). apply css rules only if those conditions are true


sytax -
 @media (condition) {

	selector {pros:values}

}
