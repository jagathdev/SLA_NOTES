üåü Types of CSS (Cascading Style Sheets)
CSS is used to style HTML documents. There are 3 main types of CSS:

‚úÖ 1. Inline CSS
Definition: Style is written directly inside the HTML tag using the style attribute.


Usage: Applies styles to a single element only.


Advantage: Quick and easy for small styling.


Disadvantage: Not reusable, not maintainable for large projects.


üîπ Example:
html
CopyEdit
<h1 style="color: red; font-size: 24px;">Inline Styled Heading</h1>


‚úÖ 2. Internal CSS
Definition: CSS rules are written inside the <style> tag in the <head> section of the HTML document.


Usage: Useful when styling a single HTML file.


Advantage: Keeps styles in one place for the page.


Disadvantage: Doesn‚Äôt apply across multiple pages.


üîπ Example:
html
CopyEdit
<head>
  <style>
    h1 {
      color: blue;
      font-size: 30px;
    }
  </style>
</head>
<body>
  <h1>Internal Styled Heading</h1>
</body>


‚úÖ 3. External CSS
Definition: CSS rules are written in a separate .css file, and linked to the HTML using <link>.


Usage: Ideal for large projects or multiple webpages.


Advantage: Reusable, easier to maintain and edit.


Disadvantage: Requires additional file loading.


üîπ Example:
HTML file (index.html):
html
CopyEdit
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <h1>External Styled Heading</h1>
</body>

CSS file (styles.css):
css

h1 {
  color: green;
  font-size: 32px;

What is CSS?
Cascading Style Sheets (CSS) is a style sheet language used to specify the presentation, layout, and styling of a document written in a markup language like HTML.
Syntax:
selector {
  property: value;
}


Why is CSS Important?
In web development, CSS is essential to style and layout web pages. It allows developers to:
Control the appearance of elements


Make websites visually appealing


Enable responsive designs that adapt to screen sizes


Note: SCSS is a preprocessor for CSS that adds features like variables and nesting.

Where is CSS Applied?
.html, .js, .jsx files (for styling within HTML and JavaScript/React)


.css files (external stylesheets)


React (.js, .jsx) supports CSS via inline styles, styled-components, or modules


Note: .css is used by .html or components in .js/.jsx

Main Purpose of CSS
Styling (colors, fonts, sizes)


Layout (positioning, spacing)


Responsive Design (adapting to devices)



Types of CSS
Inline CSS
 Directly inside an HTML element.

 <h1 style="color:red; font-size:24px;">Hello CSS</h1>


Internal CSS
 Inside the <style> tag in the <head> of an HTML document.

 <style>
  h1 {
    color: blue;
    font-size: 30px;
  }
</style>


External CSS
 Linked from a separate .css file.

 h1 {
  color: green;
  font-size: 28px;
}
 <link rel="stylesheet" href="style.css">



CSS Syntax
Selector: Targets the HTML element


Property: The style attribute (e.g., color, font-size)


Value: The assigned value for the property


p {
  color: blue;
  font-size: 16px;
}


CSS Selectors
Universal Selector: * ‚Äî selects all elements


Class Selector: .classname ‚Äî selects elements with specific class


ID Selector: #idname ‚Äî selects element with specific ID


Tag Selector: tagname ‚Äî selects by element type (e.g., h1, p)


Group Selector: selector1, selector2 ‚Äî applies same styles to multiple selectors


Multi Selector: tag.class#id ‚Äî more specific targeting


Pseudo-class Selector: :hover, :active, etc.


Pseudo-element Selector: ::before, ::after
