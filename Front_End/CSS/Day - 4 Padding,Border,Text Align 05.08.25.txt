tasks
1. Block Level Elements Alignment
Task:

Create two <div> elements (red & blue background).

Set them to display: block and check how they appear in separate lines.

Change one of them to inline and see what happens.

2. Inline Elements Styling
Task:

Take <span> tags inside a paragraph.

Set display: inline and change background color.

Try adding width and height — observe why it’s not applying (interview point).

3. Inline to Block Conversion
Task:

Create an <a> tag (link).

Change its display from inline to block and set width: 200px; height: 50px;

Observe the clickable area difference.

4. Inline-Block Navigation Menu
Task:

Create a horizontal menu using <li> tags.

Set display: inline-block so that width/height/margin work but items stay inline.

5. Image & Text Inline-Block
Task:

Place an <img> and <p> side-by-side using inline-block.

Make both vertically aligned using vertical-align: middle;.

6. Block Center Alignment
Task:

Make a <div> box with display: block, width 300px, height 200px.

Center it horizontally using margin: auto.

7. Button Inline Arrangement
Task:

Create 3 <button> elements.

Arrange them side-by-side using inline and inline-block.

Compare margin behavior.

8. Card Layout using Inline-Block
Task:

Create 3 small card divs with inline-block display.

Each should have border, padding, margin.

Show how they wrap to next line when width is small.

9. Block vs Inline Form Elements
Task:

Create a form with <input> and <label>.

Keep them inline by default.

Convert inputs to block elements so they appear one per line.

10. Inline-Block Image Gallery
Task:

Make a 3x3 image grid using display: inline-block without using flex/grid.

Add borders and spacing.

2. Margin Experiment
Task:

Create two <div> boxes stacked vertically.

Add top margin to the second box.

Notice margin collapsing in vertical direction.

3. Padding Effect
Task:

Create a box with text.

Add padding: 20px and see how text moves away from the border.

4. Different Padding on Each Side
Task:

Apply padding-top: 10px; padding-right: 30px; padding-bottom: 50px; padding-left: 5px;.

Observe how each side behaves differently.

5. Border Styles
Task:

Try solid, dotted, dashed, double borders.

Compare looks.

6. Box Sizing: Border-Box
Task:

Set box width to 200px, padding 20px, border 10px.

Without box-sizing: border-box, check size.

Then add it and see difference.

7. Nested Box Model
Task:

Create a box inside another box.

Apply margin, padding, border to both.

See how spacing changes.

8. Circle Box
Task:

Create a 200px square div with border.

Add border-radius: 50% to make it a circle.

Adjust padding and see effect.

9. Shadow & Box Model
Task:

Create a card with border, padding, and box-shadow.

Adjust padding/margin to see how shadow spacing changes.

10. Inline-Block + Box Model

Create two inline-block boxes.
