What is Bootstrap? 
1. Bootstrap is a popular Frontend Framework for creating responsive, mobile-first websites. 

2. Bootstrap offers a set of CSS utility and component classes, so you focus on structure, not CSS rules. 

3. You can integrate Bootstrap via CDN (Content Delivery Network) or by downloading and linking CSS/JS files offline. 

4. Use Bootstrap for layout, design, color, utilities, components, and more, just by applying class names.

5. Syntax: <tag class="class1 class2">Element</tag> 

Bootstrap Key Principle: Apply pre-made classes directly to elements to get instant styles and layouts!
