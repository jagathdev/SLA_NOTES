TAGS :-
A Tag is a keyword in anhle brackets <> used to define parts of webpage.

Example:
<h1> This is a heading</h1>

Elements :-
An Element is the complete set of Opening tag and content and closing tag is called a element

Example :-
<p> This complete set is a Element </p>


1. Unique tags or One Time Use tags :-
These are tags that should apper only once per page (mostly for structure or identity)

1. <html> - Root tag
2. <head> - Title and Meta info, links
3. <title> - page title (it shows browser tab)
4. <body> - page content
5. <h1> - Main Heading

the above 5 tags is used only one time in one webpage.

MOSTLY USED HTML TAGS :-

Semantic Tags

| Tag         | Use Case                            |
| ----------- | ----------------------------------- |
| `<header>`  | Top section (logo, nav)             |
| `<nav>`     | Navigation bar                      |
| `<main>`    | Main content                        |
| `<section>` | Group related content               |
| `<article>` | Blog posts, news, independent items |
| `<footer>`  | Page/footer info                    |
| `<aside>`   | Sidebar, ads                        |
| `<form>`    | Input forms                         |
| `<table>`   | Displaying tabular data             |

Non-Semantic Tags (Still commonly used)


| Tag      | Use Case                        |
| -------- | ------------------------------- |
| `<div>`  | Generic container (very common) |
| `<span>` | Inline styling container        |
| `<br>`   | Line break                      |
| `<hr>`   | Horizontal rule/divider         |
