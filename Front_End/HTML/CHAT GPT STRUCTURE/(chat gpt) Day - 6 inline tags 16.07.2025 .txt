MOSTLY USED INLINE TAGS

| Tag        | Purpose / Usage                                         |
| ---------- | ------------------------------------------------------- |
| `<a>`      | Creates hyperlinks (very commonly used)                 |
| `<span>`   | Generic inline container for styling (used with CSS/JS) |
| `<img>`    | Embeds an image                                         |
| `<strong>` | Semantically important text (usually bold)              |
| `<em>`     | Emphasized text (usually italic)                        |
| `<b>`      | Bold text (visual only, no emphasis)                    |
| `<i>`      | Italic text (visual only)                               |
| `<u>`      | Underlined text                                         |
| `<br>`     | Line break                                              |
| `<input>`  | Single-line input field                                 |
| `<label>`  | Label for form elements                                 |
| `<small>`  | Smaller font text                                       |
| `<code>`   | Displays code snippets (monospaced font)                |
| `<mark>`   | Highlights text                                         |
| `<sub>`    | Subscript text (e.g., H<sub>2</sub>O)                   |
| `<sup>`    | Superscript text (e.g., X<sup>2</sup>)                  |
