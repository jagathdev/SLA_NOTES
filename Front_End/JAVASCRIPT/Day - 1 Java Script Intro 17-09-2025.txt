1. What is JavaScript? Why do we use it? Where is it used?

Answer:
JavaScript is a high-level programming language that adds logic and interactivity to web pages. We use it to validate forms, update content without reloading, and make websites dynamic. It is used in front-end for UI, back-end with Node.js, and also in mobile/desktop apps and simple games.

2. What is variable declaration?

Answer:
Variable declaration means creating a variable and telling the program it exists.
Example:

let age;

3. What is initialization?

Answer:
Initialization means assigning the first value to a variable.
Example:

let age = 25;

4. What are data types in JavaScript?

Answer:
Data types define the type of value stored in a variable. The main ones are:

Number â†’ let x = 10;

String â†’ "Jagath"

Boolean â†’ true / false

Null â†’ null

Undefined â†’ declared but not assigned

Object â†’ { name: "Jagath", age: 23 }

Array â†’ ["red", "blue"]

5. Difference between primitive and non-primitive data types?

Answer:

Primitive: Simple values like number, string, boolean, null, undefined, bigint, symbol. Stored by value. Example:

let age = 30;


Non-Primitive: Complex values like objects, arrays, functions. Stored by reference. Example:

let person = { name: "Jagath", age: 23 };
