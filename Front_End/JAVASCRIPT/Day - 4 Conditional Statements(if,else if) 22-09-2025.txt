âœ… Truthy & Falsy Values in JavaScript
ðŸ”¹ What are Truthy & Falsy values?

In JavaScript, any value can be coerced (converted) into a boolean (true / false) when used in a condition (if, while, &&, ||, etc.).

Truthy: Values that become true when converted to boolean.

Falsy: Values that become false when converted to boolean.

ðŸ”´ Falsy Values in JavaScript

Only 7 values are considered falsy in JS:

false â†’ Boolean false itself

0 â†’ Zero

-0 â†’ Negative zero

0n â†’ BigInt zero

"" â†’ Empty string (single, double, or backtick quotes)

null

undefined

NaN â†’ Not a Number

ðŸ‘‰ Example:

if (0) {
   console.log("Truthy");
} else {
   console.log("Falsy");
}
// Output: Falsy

ðŸŸ¢ Truthy Values in JavaScript

Everything other than the falsy values is truthy.
Some examples:

true

Non-zero numbers (1, -5, 3.14)

Non-empty strings ("hello", " ", "0")

Arrays ([])

Objects ({})

Functions (function() {})

Dates (new Date())

ðŸ‘‰ Example:

if ("0") {
   console.log("Truthy");
} else {
   console.log("Falsy");
}
// Output: Truthy

âš¡ Checking Truthy & Falsy

You can quickly check using Boolean(value) or double negation !!value:

console.log(Boolean(""));      // false
console.log(Boolean([]));      // true
console.log(Boolean({}));      // true
console.log(Boolean("hello")); // true
console.log(Boolean(NaN));     // false

ðŸ’¡ Interview Questions on Conditional Statements in JavaScript

1. What are conditional statements in JavaScript?

Answer:
Conditional statements are used to make decisions in code. They check a condition (true/false) and execute different blocks depending on the result.
ðŸ‘‰ Types: if, if...else, if...else if, nested if, ternary operator.

2. What is the difference between if and if...else?

if â†’ Executes a block only if condition is true.

if...else â†’ Provides an alternative block when condition is false.

Example:

if (n > 0) console.log("Positive");
else console.log("Negative");

3. When should we use if...else if...else instead of nested if?

if...else if...else â†’ Better when you have multiple separate conditions.

nested if â†’ Used when decisions depend on multiple levels of conditions.

4. What is the ternary operator in JavaScript?

Answer:
The ternary operator (condition ? expr1 : expr2) is a shorthand for if...else.
ðŸ‘‰ It is best for one-line conditions.

Example:

let age = 20;
let status = (age >= 18) ? "Adult" : "Minor";

5. Can we nest ternary operators?

Answer: Yes, but it reduces readability.

Example:

let marks = 85;
let grade = (marks >= 90) ? "A+" : (marks >= 75) ? "A" : "B";
console.log(grade); // A

6. What are truthy and falsy values in JavaScript?

Answer:

Falsy values (7 only): false, 0, -0, 0n, "", null, undefined, NaN

Truthy values: Everything else (like "0", [], {}, " " etc.).

7. How many falsy values are in JavaScript?

Answer: 7 values â†’ false, 0, -0, 0n, "", null, undefined, NaN.

=== â†’ Strict equality (checks value + type).

Example:

9. What is the use of typeof in JavaScript conditions?

Answer:
typeof is used to check the type of a variable.

ðŸ‘‰ Useful in conditions to validate inputs.

&& (AND) â†’ Returns true only if all conditions are true.

Example:

Answer:
JavaScript automatically converts it to a boolean using truthy/falsy rules.

Example:
